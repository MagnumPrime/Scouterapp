# Scouter App

> Scouter app built with Nativescript Vue

## Usage
``` bash
# Install the latest NativeScript CLI
npm install -g nativescript@latest

# Install dependencies
npm install

# Build for Preview
tns preview --bundle

# Setup local ios/android build
tns doctor 

# Build for ios
tns run ios --bundle

# Build for android
tns run android --bundle
```
